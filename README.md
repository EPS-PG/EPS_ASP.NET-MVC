# EPSPG ASP.NET MVC

Official ASP.NET MVC library for integrating **EPSPG (Easy Payment System)** Payment Gateway.

## Features
- Sandbox & Live mode support  
- Payment initialization  
- Callback & IPN handling  
- Clean MVC-friendly implementation  

## Requirements
- .NET Framework 4.6+ or .NET 6+
- ASP.NET MVC
